## 什么是DesktopPetByAi

这是一个不同于以往桌面宠物的程序，她还集成了ai在里面，她不在是仅像以往的桌面宠物那样仅提供简单的言语反馈，她还能够跟你进行对话，从而缓解你平时无聊的心情，她提供的不在是视觉上的陪伴，还有言语上的。我们也在尽可能的使其更像一名人类(或者说一名陪伴者)。在这里你无聊的时候可以喂她，或者和她聊天，她也会在你工作时提供情绪价值。或许她更像是一名陪伴者，而不仅仅是一名宠物。在这里你不在觉得这是一串冰冷的代码，而是一位有感情的陪伴者。

---------------

## 她能为你干什么

在这里，你看到的将不是像以往的电子宠物那样，仅能够进行简单的互动。在这里，你将能够和她进行对话，她会像一位现实中存在的人一样，感知到外面的世界，记住你曾经和她的对话。

| 功能     | 说明                                           |
| -------- | ---------------------------------------------- |
| 智能对话 | 聊天、讲笑话、回答问题，上下文记忆让对话更连贯 |
| 任务执行 | “打开微信”、“搜索今天天气”——她真的会去做！     |
| 情绪陪伴 | 工作太久？她会提醒你休息；心情低落？她会安慰你 |

## 如何开始

1. 如果你是小白，没有太多的基础或者具备运行python代码的电脑环境可以尝试访问[DesktopPetByAi - 智能桌面宠物助手](https://cjz-wr.github.io/DesktopPetByAiWeb/)

在这里你可以清晰的获取到最新的，可以一键运行的安装包

> 注意:该项目是默认不具备ai的，如果你需要调用ai的话可以到https://open.bigmodel.cn/console/modelcenter/square中注册一个ai  
>
> key

2. 如果你具备python环境的话可以自己克隆该项目下来，然后运行(注意你要在项目的根目录打开终端来运行，否则可能无法找到requirements.txt文件)

   ```powershell
   git clone https://github.com/cjz-wr/DesktopPetByAi.git
   cd DesktopPetByAi
   pip install -r requirements.txt
   python main.py
   ```

   这样，你就安装好了必备的安装包

### 	配置你的ai

​		**该项目默认是不配置ai的**，如果你需要调用但是又不清楚可以按下面流程来配置

​		你如果需要一个免费的ai，则可以到https://open.bigmodel.cn/console/modelcenter/square上注册一个账号，去获取一个ai key,		在他们里面会有一个免费的ai可以用(注:后面的设置里面有两个模型接口选择，如果使用这个这里要选择智谱的接口)

​		你如果想要使用本地的ai模型，则需要你的本地模型支持openai api接口

​		当你获取好ai 的key这些必要的东西后，在系统托盘右键 → 打开“设置”，配置你的 AI Key(注意在里面选择你的模型接口)
