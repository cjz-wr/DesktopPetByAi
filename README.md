# DesktopPetByAi
这是一个智能桌面宠物，你可以与他对话，他也可以帮你处理打开软件等
更多功能将由你发掘

# 使用前

更加详细介绍网址:https://cjz-wr.github.io/DesktopPetByAiWeb/index.html

2.1.5版本已为正式版

该软件已经支持openai api接口，即可以调用本地部署的大模型

在使用前需要在小托盘里面打开设置配置好一系列功能后才能使用，不然无法使用
在使用前还需要到https://open.bigmodel.cn/console/modelcenter/square
下注册一个ai,更多功能请百度智谱ai开放平台接口来获取模型和对应的key
