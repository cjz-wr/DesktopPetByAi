项目介绍

项目概述

DesktopPetByAi 是一款集AI交互、MCP工具集成与个性化宠物形象于一体的开源桌面应用

核心定位

DesktopPetByAi 通过拟人化的桌面宠物界面，实现自然语言对话、实用工具调用（如12306票务查询、网络搜索）等功能，为用户提供便捷且有趣的桌面交互体验。

项目状态

项目支持MCP工具扩展，用户可通过二次开发添加自定义工具或交互逻辑，目前处于活跃开发阶段。

主要特点

  • 智能对话交流，支持与智谱AI进行自然语言交互
  • AI密钥管理机制，支持配置和保存API Key
  • 对话记忆系统，上下文关联存储在本地文件中
  • 集成多种实用工具：网络搜索、地理编码、12306票务查询
  • MCP工具集成，支持外部工具无缝集成
  • 模块化设计，便于功能扩展和定制，支持自定义工具开发
  • 提供EXE安装包和源码两种使用方式

重要注意事项

  • AI对话历史存储在ai_memory/目录下的JSON文件中，删除该文件可清除记忆
  • 不可删除demo_setting.json配置文件，否则可能导致程序无法启动
  • 保存AI Key时需确保程序对当前目录有写入权限，否则会提示保存失败
  • 项目为测试版，可能存在接口调用不稳定、功能冲突等问题，建议通过Issues反馈
  • 新版本增加了MCP功能支持，可以集成外部工具，如12306票务查询、Bing搜索等
核心功能

功能模块详解

深入了解DesktopPetByAi的各项核心功能模块

AI交互系统

智谱AI接口集成与密钥管理机制

AI密钥管理

  • 配置管理：通过lib/vertical_tab_widget.py中的save_ai_key函数管理API Key
  • 输入验证：检查用户输入的AI Key是否为空，为空则弹出警告提示
  • 持久化存储：将配置写入demo_setting.json文件，确保重启后生效

对话与记忆系统

  • 通过zhipu.py封装智谱AI接口，支持自然语言交互
  • AI对话历史存储在ai_memory/目录下，实现上下文关联
  • 重置记忆需删除ai_memory/下的JSON文件（不可删除目录本身）

桌面工具集成

网络搜索工具：调用Bing搜索接口获取实时信息，支持指定返回结果数量
地理编码工具：提供逆地理编码（经纬度转地址）能力，需遵循"经度,纬度"格式
12306票务工具：集成12306客户端能力，支持查询车票等

GUI界面与交互

主界面：包含宠物形象展示区、对话输入框、功能按钮
设置窗口：通过settingwindow.py实现，支持配置AI Key、默认工具等
资源管理：background/存放背景图，ico/存放应用图标

扩展与二次开发

新增工具：参照demo/目录示例，实现工具类并注册
界面定制：替换资源文件或修改settingwindow.py调整界面
AI模型切换：修改zhipu.py适配其他支持的AI模型
MCP集成

MCP工具集成

MCP（Model Context Protocol）是一种协议，用于扩展AI模型的能力

MCP集成系统

通过MCP协议，AI可以调用外部工具和访问本地资源

MCP工具实现

  • 12306票务查询：通过MCP实现火车票查询功能，用户可直接询问余票情况
  • Bing搜索：集成网络搜索能力，AI可以通过Bing获取实时信息
  • 本地文件访问：AI可以访问本地文件和资源，提供更多个性化服务

MCP开发指南

  • 工具定义：在[lib](file:///d:/code/python/desktop_pet/main/lib)目录下创建MCP工具类，定义工具名称和参数
  • 注册工具：将MCP工具注册到AI系统，使其可被AI发现和调用
  • 部署工具：将工具部署到MCP服务器，AI可根据需要调用

12306-MCP示例

查询车票：用户可询问"明天从北京到上海的火车票情况"
日期格式：支持具体日期（YYYY-MM-DD）和相对日期（today/tomorrow）
站点转换：自动将城市名转换为车站代码进行查询

MCP优势

扩展AI能力：让AI能够调用外部工具和API，突破模型训练数据限制
实时数据访问：获取实时信息，如天气、新闻、股票等
模块化设计：易于添加新工具，无需重新训练AI模型

安装与使用指南

详细的使用流程与注意事项

下载安装

1

获取AI Key

注册智谱AI开放平台账号，创建应用并获取API Key

2

下载安装包

从GitHub Releases下载最新版本的安装包，或克隆源代码自行构建

3

运行安装程序

双击下载的EXE文件，按照安装向导提示完成安装

4

配置应用

启动应用后，通过系统托盘图标打开设置窗口，配置AI密钥等必要信息

使用说明

1

开始对话

在输入框中输入问题，点击发送按钮或按Enter键发送

2

使用工具

在对话中提及需要使用工具的功能，如"查询明天北京到上海的火车票"

3

管理设置

右键单击系统托盘图标，选择"设置"进入配置界面

4

扩展功能

通过MCP协议，可以集成更多工具和功能，如Bing搜索、12306查询等
